Function startHandler() - (Lief Layout Manager)

Syntax (of Typescript Code)

function startHandler(label: string, start: string|number, is_a_container: Container = null): Item

Syntax (of Javascript Code)

function startHandler()

Parameters

None

Return value


Void

Discription

Liefs layout manager will just sit around and do nothing until you call this function.
Once Activated, It will:
  1. First Check to see if ManageArray is defined.
    1. If Defined, check screen size against ManageArray, and use that Layout.
      It does not care what use() was set to. (but you have to set it to something anyways *will fix*)
    2. If not, will use Layout defined by use() function
  2. Loop the Array<items>
    1. If each item used in layout, set visibile=true (and appropriate x, y, length, height).
    2. If each Item not used in layout, set visible=flase (and x, y, length, heigth to small values)
  3. Wait for resize event to repeat process

Example

    H("Master", 4,
            I("LeftSide", "50%"),
            I("RightSide", "50%")
    );
    use("Master");
    startHandler();
