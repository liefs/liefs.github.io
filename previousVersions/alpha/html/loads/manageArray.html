Array ManageArray:Array<[string,number,number]>

Syntax (of Typescript Code)

var ManageArray:Array<[string,number,number]>;

Syntax (of Javascript Code)

var ManageArray=[ ["SmallContainer", ifLengthIsLessThanThisAnd, ifWidthIsLessThanThis], // then use this one
              ["MediumContainer", ifLengthIsLessThanThisAnd, ifWidthIsLessThanThis], // then use this one
              ["LargerContainer", ifLengthIsLessThanThisAnd, ifWidthIsLessThanThis], // then use this one
              ["ExLargeContainer"], // if all else fails, use this one (must be bigger than all others!)
    ]; // uses first "valid" container - so list smallest to largest
// Setting ifLengthIsLessThanThisAnd or ifWidthIsLessThanThis to undefined or null means *any* is valid

Parameters

This is not a function. It is a simple array of arrays.

Description

Each Layout can be represented with the following:
["Id", minLength, minWidth]
for exmaple 
["Layout1", 800, 600]
This means then when the screen width is less than 800px AND width is less than 600px, use this layout 
["Layout1", undefined, 600]
This would be valid for ANY lenght, so long as the hieght is 600px or less,

Examples

So, Keeping with the above example: 
var ManageArray=[ ["SmallContainer", 800, 800], // then use this one
              ["MediumContainer", 1280, 800], // then use this one
              ["LargerContainer", 1440, 900], // then use this one
              ["ExLargeContainer"], // if all else fails, use this one (must be bigger than all others!)
    ]; // uses first "valid" container - so list smallest to largest
// Setting ifLengthIsLessThanThisAnd or ifWidthIsLessThanThis to undefined or null means *any* is valid
