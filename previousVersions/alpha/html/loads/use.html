Function use() - (Lief Layout Manager)

Syntax (of Typescript Code)

function use(ContainerId:string):void

Syntax (of Javascript Code)

function use(ContainerId)

Parameters

ContainerId
This is the Id value of the element.

Return value
Void

Description

Liefs Layout system needs a "Current Layout"
you must use("SomeLayout") before you startHandler();
You can also "FORCE" a layout with use().

If ManageArray is undefined, use("value") will persist.
IF ManageArray is defined, use("value") will only persist until
a resize event occurs. At that point, the above use("value") is
ignored, and the ManageArray is evaluated. To have your use("value") persist
change ManageArrary to include propper usage of "value"
, however as soon as a resize event
happens, this will be ignored IF ManageArray is defined
If ManageArray is undefined, use() value will persist.
