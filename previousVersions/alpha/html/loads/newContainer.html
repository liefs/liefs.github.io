Function newContainer() HC() VC() H() V() - (Lief Layout Manager)

Syntax (of Typescript Code)

function newContainer(label: string, true_is_hor: boolean, items: Item[], margin: number = 4): Container

Interfaces Used (Typescript)

interface Item {label: string;  start: string,  size: Position,  is_a_container?: Container,  el?:Element}

interface Container {label: string,  direction: boolean, items: Item[],  margin: number,  size?: Position, el?:Element,}

interface Position {label?: string, x: number, y: number, width: number, height: number,}

Syntax (of Javascript Code)

function newContainer(label, true_is_hor, items, margin)

Parameters

label
label is a string name of an Item (element Id value)

true_is_hor
Is a boolean value. If its true then the container is Horizontal. false then it is vertical

items
is an Array of Items

Return value (Typescript) Container

Description

This function simply creates, and returns a Object defined by above interfaces.

Examples

var a=newContainer("ContainerName",true,[newItem("Tree","120px"),newItem("Main","100%")], 4);
{ label: 'ContainerName',
  direction: 4,
  items:
   [ { label: 'Tree', start: '120px', size: { x: 0, y: 0, width: 0, height: 0 } },
     { label: 'Main', start: '100%', size: { x: 0, y: 0, width: 0, height: 0 } } ],
  margin: 4 }

Abbreviations

/* Short Form for "create Horizontal Container"  Good to Know - You will never use this one */
HC(id, margin, arrayOfItems){return newContainer("_"+id, true, arrayOfItems ,margin);}

/* Short Form for "create Vertical Container"    Good to Know - You will never use this one */
VC(id, margin, arrayOfItems){return newContainer("_"+id, true, arrayOfItems ,margin);}

/* rather than listing Items in an [array format] use argument list instead  -> Margin Optional */
H(id, <optional Margin>, ...arrayOfItems) /* H is short for Horizontal Container */
V(id, <optional Margin>, ...arrayOfItems) /* H is short for Vertical Container */

/* so this */
newContainer("ContainerName",true,[newItem("Tree","120px"),newItem("Main","100%")], 4)

/* can be shortened to this */
H("ContainerName", 4, /* if you want Margins 4 */
    I("Tree","120px"),
    I("Main","100%")
)

/* or even */
H("ContainerName",  /* if you want Margins Default */
    I("Tree","120px"),
    I("Main","100%")
)
