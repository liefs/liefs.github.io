Function stopHandler() - (Liefs Layout Manager)

Syntax (of Javascript Code)

function stopHandler()

Parameters

None

Return value


Void

Discription

This will stop liefs-layout-manager completly.
All "last used values" remain in DOM - further resize event ignored

Example

    H("Master", 4,
            I("LeftSide", "50%"),
            I("RightSide", "50%")
    );
    use("Master");
    startHandler();
    stopHandler();
The above code WILL do something. It will resize all elements defined as items,
Then ignore resize events, leaving all elements at their positions calculated
for initial screen size.
