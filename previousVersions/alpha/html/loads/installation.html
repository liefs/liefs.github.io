Installation

At the end of the day, Liefs layout manager is a simple Javascript file.
basic.ts
basic1.ts
basic.js
basic.min.js
basic1.js
basic1.min.js
 *Source of Liefs Layout Manager (Typescript)
identical to *Source, but every export statement has been change to standard statement (Typescript)
*Source of Liefs Layout Manager (Javascript) >tsc basic.ts To Create This file
*uglifyjs basic.js > basic.min.js - uglified version
*export free Liefs Layout Manager (Javascript)
*uglifyjs basic1.js > basic1.min.js - uglified version
*.ts (Typescript), *.js (Javascript)

Instalation

Option #1 - Easy - None

Officially, You don't have to install anything to make Liefs Layout Manager work for you.
Just include http://liefs.paperplane.io/js/basic1.js in your html file.

See The Following Code Work As A Page
(View Source After You Click Above Link) 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <script src="http://liefs.paperplane.io/js/basic1.js"></script> <!-- this is the REQUIRED CODE for liefs to work! -->
    <title>Example 00 - Bare Bones</title>
</head>
<body style="color:white">

<div id="Top" style="background-color: red">"Top": Resize Screen To See Effect</div>
<div id="LeftSide" style="background-color: blue">"LeftSide"</div>
<div id="RightSide" style="background-color: green">"RightSide": View Source Of Page - Its That Easy</div>

</body>
</html>
<script>
    window.onload = function () {
        marginDefault=4;
        V("Master",
                I("Top", "200px"),
                h("Bottom", "100%",
                        I("LeftSide", "30%"),
                        I("RightSide", "70%")
                )
        );
        use("Master");
        startHandler();
    }
</script>

Option #2 Npm

This will add Lief Layout Manager To your Project 
npm install liefs-layout-manager --save
you will find:
./node_modules/liefs-layout-manager/basic.js
./node_modules/liefs-layout-manager/basic.min.js
./node_modules/liefs-layout-manager/basic.ts
./node_modules/liefs-layout-manager/basic1.js
./node_modules/liefs-layout-manager/basic1.min.js
./node_modules/liefs-layout-manager/basic1.ts
./node_modules/liefs-layout-manager/package.json
./node_modules/liefs-layout-manager/README.md

Typescript 
import {I, H, V, h, v, newItem, newContainer, use,
    startHandler, ManageArray, update, checkWinWH} from './node_modules/liefs-layout-manager/basic';

marginDefault=4;
V("Master",
    I("Top", "200px"),
    h("Bottom", "100%",
        I("LeftSide", "30%"),
        I("RightSide", "70%")
    )
);
use("Master");
startHandler();
Javascript 
var liefs=require("./node_modules/liefs-layout-manager/basic.js");
var I=liefs.I,V=liefs.V,H=liefs.H,v=liefs.v,h=liefs.h,newItem=liefs.newItem,newContainer=liefs.newContainer;
var use=liefs.use,startHandler=liefs.startHandler, ManageArray=liefs.ManageArray, update=liefs.update;
var checkWinWH=liefs.checkWinWH;

var marginDefault=4;
V("Master",
    I("Top", "200px"),
    h("Bottom", "100%",
        I("LeftSide", "30%"),
        I("RightSide", "70%")
    )
);
use("Master");
startHandler();

Option # GitHub

This will add Lief Layout Manager To your Project 
git clone https://github.com/electriclief/liefs.git
This will make a new directory called liefs with this file structure:https://github.com/electriclief/liefs
Load Typescript and Javascript like above, (For NPM) but use:

./liefs/ts instead of ./node_modules/liefs-layout-manager for the Typescript files and
./liefs/js instead of ./node_modules/liefs-layout-manager for the Javascript files


Github installation include this web site, so If you would like to view this web site locally, 
npm install -g serve
serve ./liefs -a localhost
Then click on "index.html"
