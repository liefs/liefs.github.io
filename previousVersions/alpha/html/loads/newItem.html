Function newItem() - (Lief Layout Manager)

Syntax (of Typescript Code)

function newItem(label: string, start: string|number, is_a_container: Container = null): Item

Interfaces Used (Typescript)

interface Item {label: string;  start: string,  size: Position,  is_a_container?: Container,  el?:Element}

interface Container {label: string,  direction: boolean, items: Item[],  margin: number,  size?: Position, el?:Element,}

interface Position {label?: string, x: number, y: number, width: number, height: number,}

Syntax (of Javascript Code)

function newItem(label, start, /*optional*/ is_a_container)

Parameters

label
label is a string name of an Item (element Id value)

start
Start is either a constant value (in pixels) ie: "120px" or 120,
or a percentage of the Container (after constants assigned) ie "50%" or "100%"

is_a_container
Use this value ONLY if the Item, is also to be used as a Container. If so, the container goes here

Return value
(Typescript) Item

Description

This function simply creates, and returns a Object defined by above interfaces.

Examples

var w=newItem("Tree", 120);
{ label: 'Tree',
  start: '120px',
  size: { x: 0, y: 0, width: 0, height: 0 } }

var x=newItem("Main", "120px");
{ label: 'Main',
  start: '120px',
  size: { x: 0, y: 0, width: 0, height: 0 } }

var y=newItem("Header", "100%");
{ label: 'Header',
  start: '100%',
  size: { x: 0, y: 0, width: 0, height: 0 } }

var z=newItem("Middle","100%", newContainer("ContainerName",true,[w,x,y], 4)); /* see Container Docs */

Abbreviations

/*  'newItem' has been shortened down to 'I' so ... */
var x=newItem("Tree","200px");  /* and */
var x=I("Tree","200px");        /* are equvilent */

/* Rather than making a container, then an Item wrapper, you can combine them like this */
h(id, start, <optional margins>,...arrayOfItems) /* Item with (Create Horizontal Container) */
v(id, start, <optional margins>,...arrayOfItems) /* Item with (Create Vertical Container) */

/* So instead of writing this */
var hCont = newContainer("_HorCont", true , [I("Tree","120px"),I("Main","100%")], 4);
var hItem = newItem("HorCont", "100%", hCont);

/* It can be shortened to this <They are IDENTICAL> */
var hCont = h("HorCont","100%", 4,  /* to use Margin 4 */
    I("Tree","120px"),
    I("Main","100%")
)
var hCont = h("HorCont","100%",     /* Use Default Margin */
    I("Tree","120px"),
    I("Main","100%")
)
