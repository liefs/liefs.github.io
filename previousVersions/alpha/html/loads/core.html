Liefs Layout Manager - Core

This Project is devided into sections:
  1. Core
    1. newItem()
    2. newContainer()
    3. update()
  2. Handler
  3. Directive
  4. Misc

Core is the "evaluating" part of the project. It is pure logic. It doesn't even know the dom exists. Simply put, this is a fancy evaluating function transforming input into output.

The Input is the form of Items and Containers.

Every Page Layout must have at least One Outer Container, with at least one Item in it.

Lets Say we want:
Blue to be a fixed width of 200px,
and Green to fill the rest of the width.

Red to be a fixed height of 220px,
and Blue and Green to fill the rest of the height.

First load/install the javascript code for Liefs Layout Manager.
Then Define the Layout (Javascript):
> var o=V("Outer", 4, // using 4px margins...
            I("Red","220px"),
            h("Inner","100%",
                I("Blue","200px"),
                I("Green","100%")
            )
        );
And use Core to Evaluate where the Co-Ordinates of the windows are: 
> update(1920,1080, o);
{ Red: { x: 4, y: 4, width: 1912, height: 220 },
  Blue: { x: 4, y: 228, width: 200, height: 848 },
  Green: { x: 208, y: 228, width: 1708, height: 848 },
  Inner: { x: 4, y: 228, width: 1912, height: 848 } }
