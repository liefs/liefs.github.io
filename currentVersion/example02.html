Example 01 This is a basic example for Liefs Layout Manager.

This page (see view source this page) demonstrates a simple layout.
It consists of x2 DIV tags: id="Menu" and id="right"

This is the COMPLETE Javascript Code required for Liefs Layout Manager to Map this Page. 
h("example01", 4 ,
  I("Menu","200px", "150px", "250px"),
  I("ex01", "100%")
);
line:1 - Lief's Layout Manager function "h" defines a Horizontal Item, named "Master" with 4px Margins.

line:2 - Lief's Layout Manager function "I" defines an Item, named "Menu". For this statement to be valid, there must be exactly one matching selector (default is id ...so matching id) or an error will be thrown. This "Menu" Item will be 400px Wide, complete with dragBar stretching to a minimum value of 300px to a maximum value of 500px.

line:3 - Lief's Layout Manager function "I" defines an Item, named "right". For this statement to be valid, there must be exactly one matching selector (default is id ...so matching id) or an error will be thrown. This "right" Item will be 100% of the pixels left, after all others are subtracted from available.

line:4 close brackets, end line

Note: Liefs Layout Manager must be loaded with a script tag. 
<script src="...js">
Note: Liefs Layout Manager REQUIRES basic css styling of dragBars. Customize to suit, but they must be position: fixed to work. 
  .Hdragbar {position: fixed;cursor: col-resize;border-style: none;border-width: 0px;border-radius: 0px;}
  .Vdragbar {position: fixed;cursor: row-resize;border-style: none;border-width: 0px;border-radius: 0px;}

Example 02 - You can use/reuse any Item, in Any Layout

This example demonstrates that you can show any Item, in Any Layout.
In this example, we are repeating our Menu Item on the left,
and our Example 01 below. (Complete with dragBar)

This is the entire JavaScript require to Manage This Example. 
  h("example02", 24 ,
      I("Menu","200px", "200px", "600px"),
      v("ex02v", "100%", 24,
        I("ex02", "400px","300px", "700px"),
        I("ex01", "100%")
      )
  );
