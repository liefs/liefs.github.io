Lief's Layout Manager is JavaScript code that organizes content on your screen/display

Lief's Layout Manager has two main functions.
  1. Control the positioin of Elements (Usually <div>'s) On the screen, using a hiarchy of Items.
  2. Control which Items will be displayed under which conditions.
This process is best understood by examples and a few short videos (playlist)

Once you have loaded liefs-layout-manager, you have access to the following Objects:
<script src="liefs-layout-manager.V?.?.?.min.js"></script>
// Check npm for latest version

> Item
// Liefs's Layout Manager Item Class
// Element/(Group of Elements)
Dragbar 
> Dragbar
// Liefs's Layout Manager Dragbar Class
// Optional Dragbar for above
Container 
> Container
// Liefs's Layout Manager Container Class
// Used to Group Item (s)
Layout 
> Layout
// Liefs's Layout Manager Layout Class
// Root Item, with Conditional Function for usage
Handler 
> Handler
// Liefs's Layout Manager Handler Class
// Chooses Layout by Conditional Function

> Handler.verbose = false;  // This turn off feeback on Console.
> Handler.delayUntilStart = 200;  // This milli-second start delay can be changed
> Handler.callback = function() { /* Do Something after Lief's does a refresh */ };

> Handler.handlers[0];  // gives you access to (Main - usually only) Handler instance
> Handler.resizeEvent();  // Manually Trigger a Lief's layout Manager Page Refresh
> Handler.resizecallbackthrottle = 1000;  // Manually force delay between refreshs (Default 0)
> Item.selector = function () { return {"#" + this.label}}   // Selectors Do not HAVE to be id
> Handler.handlers[0].label;              // there is a label attribute for all Lief's Objects
> Handler.activeContainer.label;          // there is a label attribute for all Lief's Objects
> Container.marginDefault = 8;            // Sets Default Margin size (Default = 4(px))
> Container.of( Item.get("SomeExistingItem") ); // Find Current Parent of Item (Changes with Layout)
> Handler.activeContainer.update(1920, 1080); Handler.activeContainer.lastupdate;
// Force test screen size to see position results
> Handler.activeContainer.lastupdate;     // Returns Object Key=Item name, Value = Current Co-ordinates of Item
> Handler.handlers[0].label;              // there is a label attribute for all Lief's Objects
> Handler.activeContainer.label;          // there is a label attribute for all Lief's Objects
> Item.get("SomeDefinedItem").label;      // there is a label attribute for all Lief's Objects
> Handler.handlers[0].iterator();         // there is an iterator function for all Lief's Objects
> Handler.activelayout.iterator();        // there is an iterator function for all Lief's Objects
> Handler.activeContainer.iterator();     // there is an iterator function for all Lief's Objects
> Item.get("SomeDefinedItem").iterator(); // there is an iterator function for all Lief's Objects
> Handler.handlers[0].toString();         // there is a toString() for all Liefs's Objects
> Handler.activeContainer.toString();     // there is a toString() for all Liefs's Objects
> Item.get("SomeDefinedItem).toString();  // there is a toString() for all Liefs's Objects
> Item.keys();                            // Returns Array of all Item Labels
> Item.get("ItemDefinedWithPages").currentPage; // find Array index here
